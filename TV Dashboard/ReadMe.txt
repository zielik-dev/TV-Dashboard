In scope:
---------
SQL Server, EF Core, Bootstrap 5

High Level Description:
-----------------------
This simple ASP.NET Core application in MVC design pattern created only as TV Dashboard display that reads data from single database table and present the content in an internet browser.
It uses Entity Framework Core 6.0, SQL Server and Bootstrap 5 classes.
